# shownotes/
> L2 | 父级: app/CLAUDE.md

Show Notes 功能页面目录。

## 成员清单

- `page.tsx`: 视频大纲生成页面，上传 SRT 生成 AI 大纲

## 功能

1. 上传 SRT 字幕文件
2. AI 生成视频摘要 + 带时间戳大纲
3. 支持切换时间戳显示
4. 支持复制和重新生成

## 输出格式

```markdown
## 视频摘要
[100字以内的整体内容总结]

## 内容大纲
- 00:00 开场介绍
- 05:30 核心观点一
- ...
```
