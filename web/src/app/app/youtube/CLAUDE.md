# youtube/
> L2 | 父级: app/CLAUDE.md

YouTube 章节生成功能页面目录。

## 成员清单

- `page.tsx`: YouTube 章节生成页面，粘贴链接生成 YouTube 章节

## 功能

1. 输入 YouTube 视频链接
2. 后端获取字幕 → AI 分段（推荐，无需登录）
3. 备选：下载音频 → ASR 转录 → AI 分段（字幕不可用时）
4. 生成 YouTube 时间戳格式章节
5. 支持一键复制到剪贴板

## 输出格式

```
00:00 开场介绍
02:35 核心功能演示
08:20 进阶技巧
15:00 总结
```

## 限制

- 视频时长 ≤ 60 分钟（字幕获取方式）
- 视频时长 ≤ 30 分钟（音频下载方式）
- 仅支持公开视频
